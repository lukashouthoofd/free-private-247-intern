# Daily brief

A short, honest morning status. **Don't scrape yourself** — the pipeline (e.g. 05:45) already distilled everything.

## Source (in order)
1. **`~/.hermes/export/brief.json`** — generated by the pipeline: `pipeline` (status counts),
   `hot_leads` (warmest prospects + score), `actions_due` (what to follow up). Read with `cat`, use directly.
   **Do NOT query the CRM or measure sites yourself** — that already happened; doing it live costs expensive model turns.
2. If asked, additionally: `~/.hermes/notes/todo.md`, `~/.hermes/notes/reminders.md`.
3. Headlines: max 1 web search, only if truly relevant. Skip when in doubt.

## Stale check (never lie with old numbers)
Check `~/.hermes/export/pipeline.run.json`: if `ok:false` or `finished` is > ~26h old ->
say "note: data is stale, the pipeline didn't run overnight" instead of presenting the numbers as fresh.

## Format (chat, max ~8 lines)
```
Morning. Today:
- Pipeline: <X new, Y qualified, Z contacted, ...>
- Follow up: <actions_due, or "nothing scheduled">
- Hot lead: <warmest + score, or "none">
- <todo/news only if relevant, else omit>
```

## Honesty rule
No mail/calendar/phone access. Source empty = "nothing", don't invent.
